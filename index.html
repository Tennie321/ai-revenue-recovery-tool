Frequently Asked Questions

What is the R.E.C.O.V.E.R. framework?

A 7-step methodology: Research the loss reason, Evaluate recovery potential, Craft a win-back strategy, Orchestrate outreach, Verify engagement, Execute with incentives, and Retain with renewed terms.

Do I need to install anything?

No! The tool uses only Python's standard library — zero external dependencies. Just download and run with Python 3.8+.

How is the recovery score calculated?

The score (0-100) combines: recovery probability (baseline), minus time penalty (3 pts/month since loss), plus value bonus (1 pt per $5K prior annual value), plus relationship bonus (0-25 pts depending on key contact strength).

Can I import my own client data?

Yes. Use batch mode with a JSON file containing your lost client data. The tool processes unlimited clients and generates grouped reports with loss reason breakdowns.

What output formats are available?

HTML (dark-themed dashboard), Markdown (documentation-ready), CSV (spreadsheet), and JSON (machine-readable). Generate all four at once or select specific formats.
